## 首次使用：配置你的知识库

### 第一步：获取你的 API 凭证

1. 打开 Get 笔记网站：https://www.biji.com/subject
2. 进入你想要使用的知识库
3. 点击顶部的「API 设置」按钮
4. 复制你的 **API Key** 和 **知识库 ID**

### 第二步：告诉 Claude 配置你的知识库

**你只需要在对话框中这样说：**

```
帮我配置 Get 笔记：
- 知识库名称：政经参考
- API Key: 你的_API_KEY
- 知识库 ID: 你的_ID
- 设为默认
```

或者更简单：

```
配置我的 Get 笔记：政经参考，API Key 是 xxx，知识库 ID 是 xxx，设为默认
```

**✅ 配置成功后，你就可以开始使用了！这个配置只需要做一次。**

## 搜索和提问

配置完成后，你可以直接开始提问：

### 示例对话

**你说：**
```
搜索我的 Get 笔记：2025年房产投资趋势
```

**或者：**
```
在我的 Get 笔记里查一下机器学习算法
```

**或者：**
```
帮我从 Get 笔记中找一下关于 Python 最佳实践的内容
```

**Claude 会：**
1. 自动调用 API 搜索你的知识库
2. 显示 AI 处理后的答案
3. 保存结果到本地 Markdown 文件（文件名：`get_知识库名_时间.md`）

## 追问（自动记住上下文）

**重要：你不需要重复之前的问题，Claude 会自动记住上下文！**

### 示例对话

**你：** 搜索我的 Get 笔记：什么是微服务架构？

**Claude：** [返回答案并保存到 `get_工作笔记_20260125_143022.md`]

**你：** 它有什么优缺点？

**Claude：** [自动带上前面的上下文继续回答，并追加到同一个文件中]

**你：** 有哪些公司在用？

**Claude：** [继续追加到同一个文件]

**✅ 同一个话题的所有问答都会保存在同一个文件中，方便你后续查看！**

## 控制引用显示

引用（refs）是指 AI 回答时引用的原始笔记来源。

### 全局设置（推荐）

**开启引用（默认）：**
```
开启全局引用显示
```

**关闭引用：**
```
关闭全局引用显示
```

**✅ 这个设置会一直保持，直到你再次修改**

### 单次搜索时指定

**这次搜索要显示引用：**
```
搜索我的 Get 笔记（要引用）：人工智能
```

**这次搜索不要引用：**
```
搜索我的 Get 笔记（不要引用）：快速查一下定义
```

## 控制深度思考

深度思考模式会让 AI 进行更深入的分析，但会稍微慢一些。

### 默认情况

**深度思考默认是开启的**，适合复杂问题。

### 关闭深度思考

对于简单问题，你可以关闭深度思考来加快速度：


**或者：**
```

```

## 常见问题

### Q1: 我需要每次都提供 API Key 吗？

**不需要！** 你只需要在首次配置时提供一次，之后 Claude 会自动记住。

### Q2: 追问时需要重复之前的问题吗？

**不需要！** Claude 会自动记住同一会话中的所有问答历史。

### Q3: 如何知道什么时候会创建新文件？

**以下情况会创建新文件：**
- 你明确说「开始新会话」或「新话题」
- 你切换到不同的知识库
- 你重启了 Claude Code

**否则会追加到同一个文件中。**

### Q4: 生成的文件保存在哪里？

**当前工作目录**。每次搜索会生成两个文件：
- `get_知识库名_时间.md` - 问答记录
- `get_知识库名_时间_引用.md` - 引用来源记录

### Q5: 一个 API Key 能用于多个知识库吗？

**可以！** 一个 API Key 可以访问你账户下的所有知识库，只需要提供不同的知识库 ID。

### Q6: 如何删除某个知识库配置？

**你说：**
```
删除「学习笔记」知识库配置
```

### Q7: 如何清空会话历史？

**你说：**
```
查看所有会话
```

然后：
```
清空会话 xxx（会话ID）
```

## 💡 使用技巧

### 1. 让 Claude 自动判断

你不需要记住所有命令格式，直接用自然语言说就行：

**好的示例：**
- "帮我搜一下 Get 笔记里关于 xxx 的内容"
- "继续问：那么 yyy 呢？"
- "新话题：查一下 zzz"
- "切换到技术笔记，搜索 aaa"

**Claude 会自动识别你的意图！**

### 2. 合理使用文件累积功能

同一个话题的多次追问会保存在同一个文件中，方便你：
- 完整回顾整个思考过程
- 导出成笔记或报告
- 分享给他人

### 3. 合理设置全局引用

如果你经常需要查看来源，就开启全局引用；
如果你只想要快速答案，就关闭全局引用。

### 4. 为不同用途创建不同知识库

例如：
- 工作笔记（默认）
- 技术学习
- 个人成长
- 项目资料
