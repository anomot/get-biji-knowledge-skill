# Get-Biji-Knowledge-Skill: 你的本地 AI 知识引擎

> 让 Claude 拥有你的私人知识库，打通 Get 笔记与本地工作流。

## 🌟 简介

**Get-Biji-Knowledge-Skill** 是一个连接 **Get 笔记 (biji.com)** 与 **本地开发环境** 的桥梁。它让 Claude Code 能够直接读取、理解并引用你在 Get 笔记中积累的高质量知识，将其转化为本地的永久资产。

不再受限于网页端的封闭生态，现在，你可以在任何支持 Claude Code 的地方（终端、VS Code 及相关衍生 AI IDE、Obsidian）随时调用你的“第二大脑”。

## 💎 为什么选择它？

相比于直接在 Get 笔记网页版使用 AI，本工具提供了本质的区别：

| 维度 | Get 笔记网页版 AI | Claude Code + 本工具 |
| :--- | :--- | :--- |
| **生态开放性** | ❌ **封闭生态**：内容锁在网页里，**只能单篇导出**，无法批量导出或与其他工具联动 | ✅ **开放集成**：支持 Obsidian、**VS Code/相关衍生 AI IDE**、终端，知识随处调用 |
| **数据归属** | ⚠️ **平台留存**：可用自己的大模型二次加工并保存到知识库，但**加工内容依然在网站中** | ✅ **本地沉淀**：生成内容直接保存在本地，**文档可以拿到任何地方使用**，完全掌控 |
| **AI 能力** | ⚠️ **模型受限**：依赖平台特定模型 | ✅ **顶级模型**：利用 Claude 4.5 等顶级模型，逻辑更强，质量更高 |
| **知识管理** | 🔍 **单次查询**：查完即走，难以形成体系 | 📝 **知识再造**：将碎片笔记重组为系统性文档（文章、报告），**生成带有独立索引的本地文档** |

## 🚀 你能用它做什么？

- **✍️ 深度写作与输出**
  *“搜索我关于‘创新管理’的所有笔记，结合当前市场环境，写一篇给团队的内部管理建议书。”*
  —— 让过去的思考服务现在的创作。

- **🧠 辅助决策**
  *“综合我记录的所有关于‘宏观经济周期’的分析，推演未来 3 年的投资策略。”*
  —— 让分散的信息形成系统的判断。

- **🔗 打造第二大脑**
  在 **Obsidian** 中直接调用，生成的回答自动保存为笔记，无缝融入你的知识网络。
  *甚至可以利用其他 Skill 进一步生产 Canvas、思维导图、Excalidraw 等可视化内容。*

## 🎮 极简使用体验

### 1. 可视化配置
提供 **Web 图形界面**，无需修改代码，**支持直观设置文档输出路径**，小白也能轻松上手。

### 2. 自然语言交互
配置完成后，就像和真人聊天一样使用。

#### 场景 A：精准搜索（指定知识库）
> **用户**：在“政经参考”知识库中搜索：2025 年房地产税政策走向。
> **Claude**：[精准锁定范围 -> 深度解析 -> 生成本地报告]

#### 场景 B：智能路由（自动匹配）
> **用户**：搜索：如何提升非暴力沟通能力？
> **Claude**：[自动检测到该问题属于“心理学”知识库 -> 调取相关笔记 -> 生成建议]

#### 场景 C：联合分析（多库对比）
> **用户**：结合“历史规律”和“科技趋势”两个知识库，分析人工智能对未来就业结构的影响。
> **Claude**：[跨库提取信息 -> 对比分析 -> 输出综合研判]

#### 场景 D：全库扫描（广域搜索）
> **用户**：在所有知识库中查找关于“反脆弱”的提及。
> **Claude**：[全局扫描 -> 汇总所有相关点]

### 3. 上下文与会话管理
- **支持追问**：无需重复背景，Claude 会自动记住上一轮的搜索结果和对话内容。
- **开启新会话**：输入“新话题”或“开始新会话”，Claude 将清空上下文，重新开始。
- **清除对话历史**：明确要求“清除历史”即可重置状态。

### 4. 维护与优化
随着你在 Get 笔记中积累的内容越来越多，建议定期更新知识库的描述信息，以便 AI 能更精准地找到内容。
- **更新方式**：直接在输入框打上 **“我要更新 Get 知识库描述”**，系统会自动扫描最新内容并重新生成描述。

## 📦 快速开始

### 安装
```bash
cd ~/.claude/skills/
git clone https://github.com/anomot/get-biji-knowledge-skill.git
``````

### 初始化配置
只需要对 Claude 说一句大白话：

> **“我要添加/修改 Get 笔记知识库。”**

系统会自动引导你完成配置，或者打开可视化的配置界面供你填写。

#### 🔑 如何获取凭证？
1. 访问 [Get 笔记](https://www.biji.com/note)。
2. 点击 **知识库** -> 进入某个知识库页面。
3. 点击知识库顶部的 **"API 设置"**。
4. 点击创建 API，**生成的 API Key 只可见一次**，请务必保存好。
5. 在 API Key 生成页面中，同时找到 **知识库 ID**。

## 🔒 安全说明

- **数据隐私**：所有生成的内容均存储在您的本地设备中。
- **配置存储**：您的 API Key 和配置信息安全地存储在本地文件 `~/.claude/get-biji-knowledge-skill-config.json` 中，不会被上传。
